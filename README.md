# algol68-cli-hello-world

## Description
A POC for algol 68 programming language.

## Tech stack
- algol

## Docker stack
- ubuntu:22.04

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
